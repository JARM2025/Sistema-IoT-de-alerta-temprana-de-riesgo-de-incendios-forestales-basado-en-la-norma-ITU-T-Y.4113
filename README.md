# Sistema-IoT-de-alerta-temprana-de-riesgo-de-incendios-forestales-basado-en-la-norma-ITU-T-Y.4113
Sistema IoT modular de alerta temprana de riesgo de incendios forestales, basado en la norma ITU-T Y.4113. Incluye sensores ESP32 (DHT22 y anemómetro RS485), pasarela Raspberry Pi con InfluxDB y Grafana, backend Node.js con API pública, dashboard web y aplicación Android para visualización remota y alertas en tiempo real.
