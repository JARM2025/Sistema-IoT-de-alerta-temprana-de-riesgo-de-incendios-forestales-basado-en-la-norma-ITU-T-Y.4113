# Sistema-IoT-de-alerta-temprana-de-riesgo-de-incendios-forestales-basado-en-la-norma-ITU-T-Y.4113
Sistema IoT modular de alerta temprana de riesgo de incendios forestales, basado en la norma ITU-T Y.4113. Implementa sensores ESP32 (DHT22, anemómetro RS485), pasarela Raspberry Pi con InfluxDB y Grafana, cálculo del índice F y alertas por SMS y API web.
